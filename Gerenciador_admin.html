Gerenciar Produtos (Localmente)

Atenção: Dados Locais!

As alterações feitas aqui são armazenadas **apenas neste navegador** e **não serão salvas na nuvem**. Se você limpar o cache do navegador, usar outro dispositivo ou navegador, os dados serão perdidos. As alterações **NÃO** serão refletidas no seu site público para outros visitantes.

Para imagens, este gerenciador armazena **apenas o caminho/URL**. Você deve **manualmente colocar os arquivos de imagem** na pasta `images/` (ou outra pasta que você especificar) ao lado dos seus arquivos HTML para que elas apareçam no site.

Funcionalidade de Download de Imagens da Web:

Ao inserir uma URL de imagem da web, um botão "Baixar Imagem" aparecerá. Ao clicar nele, a imagem será baixada para a pasta de downloads padrão do seu navegador. **Não é possível selecionar uma pasta específica no seu PC para salvar as imagens diretamente através do navegador por motivos de segurança.** Em alguns casos, o navegador pode abrir a imagem em uma nova aba em vez de baixar, dependendo das configurações do servidor de origem da imagem.
